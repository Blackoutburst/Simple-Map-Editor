# Simple Map Editor

## Controls
- Rotate Camera: **Left Shift** + **Left Mouse**
- Move Camera: **Left Shift** + **Right Mouse**
- Reset Camera: **R**
- Open File: **O**
- Save File: **S**

## File Format
SME files use the simplest format possible
```SME

```

## Building
> [!IMPORTANT]
> Java 21+ is required
```
./gradlew build
```
